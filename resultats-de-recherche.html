Résultats de recherche

Voici un exemple d'un listing de résultats de recherche :

Les résultats de recherche présentent les objets conformément à votre recherche.

Exporter :

Vous avez la possibilité d'exporter vos résultats de recherche via un bloc d'export présent en haut à droite.

Les formats d'export sont :

  • PDF : document PDF prévu pour une consultation sur ordinateur ou pour une impression
  • CSV : document CSV prévu pour une exploitation externe avec d'autres solutions applicatives utilisant ce format
  • JSON : document JSON prévu pour une exploitation externe avec d'autres solutions applicative utilisant ce format
  • EMAIL : lien permettant d'envoyer un document PDF via email

Carte maps :

La carte affichera la position de tous les objets des résultats de recherche.
Cela vous permettra de connaître la position exacte d'un objet.

Options fiche objet :

Une fiche objet permet les actions suivantes :

  • Détail fiche : permet d'accéder à la fiche de l'objet
  • Localiser : permet de localiser l'objet sur la carte
  • Favoris : permet d'ajouter l'objet à ses favoris
