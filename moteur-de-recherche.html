Moteur de recherche

Le moteur de recherche permet de faire une recherche globale sur l'ensemble des objets existants dans Tourisma.

Deux types de recherche sont proposés : une simple et une autre avancée

Recherche simple :

La recherche simple permet de faire, comme son nom l'indique, une recherche simple par nom d'objet dans ceux présents dans la base de données :

Recherche avancée :

La recherche avancée permet une recherche beaucoup plus approfondie selon les critères des objets présents dans la base de données :

Les attributs de recherche changeront selon le type d'objet sélectionné. Vous aurez par conséquent le choix de choisir les attributs avec lesquels vous souhaitez effectuer votre recherche.

Une dernière colonne permet de spécifier encore plus précisément les valeurs souhaitées pour chacun des attributs sélectionnés.
